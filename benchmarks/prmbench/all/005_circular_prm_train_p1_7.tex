\begin{theorem}
Find the integer $n,$ $-90 < n < 90,$ such that $\tan n^\circ = \tan 312^\circ.$
\end{theorem}

\begin{proof}
Step 0: So we're looking for an integer $n$ that satisfies two properties: $-90 < n < 90$ and $\tan n^\circ = \tan 312^\circ$.
Step 1: Right, and we should mention that the function $\tan$ has a period of $180^\circ$.
Step 2: That's true. That means that $\tan (x^\circ) = \tan (x^\circ + 180^\circ)$.
Step 3: Right. So to find the value of $n$ that satisfies the equation $\tan n^\circ = \tan 312^\circ$, we can add or subtract $180^\circ$ from $312^\circ$ until we find an angle $n^\circ$ that is between $-90^\circ$ and $90^\circ$.
Step 4: Now, let’s check by assuming $n = -48^\circ$ and use $-48+180 = 132^\circ$, $\tan 132^\circ = \tan 312^\circ$ to verify the initial assumption.
Step 5: Right. This satisfies both the equation and the inequality.
Step 6: Right. So the integer $n$ that we're looking for is $-48$.
\end{proof}
