\begin{theorem}
Compute $58_9 - 18_9.$ Express your answer in base $9.$
\end{theorem}

\begin{proof}
Step 0: Subtraction works the same in base $9$ as in base $10$.
Step 1: So we just find the difference of the numbers in the right column, which is $8-8=0$.
Step 2: Now we need to find the difference of the numbers in the left column, similar to how it works in base $5$. This is $5-1=6$ using base $5$ rules.
Step 3: That's our answer.
Step 4: The answer is $60_9$.
Step 5: Notice that we didn't need to convert to base $10$ to solve this problem.
\end{proof}
