\begin{theorem}
Evaluate the indefinite integral.
$\int 208 \cos^4(16x) dx$
\end{theorem}

\begin{proof}
The answer is:
\[
78 x + \frac{13 \sin{\left(16 x \right)} \cos^{3}{\left(16 x \right)}}{4} + \frac{39 \sin{\left(16 x \right)} \cos{\left(16 x \right)}}{8}
\]
\end{proof}
